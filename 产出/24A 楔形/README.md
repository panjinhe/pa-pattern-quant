# 文档摘要

- 原文把“楔形”定义得很宽：凡是带有三次推动的结构，不论是否严格收敛，都可以视作楔形的一种变体。
- 主交易语义分成两类：`楔形顶 / 楔形熊旗` 倾向向下反转，`楔形底 / 楔形牛旗` 倾向向上反转。
- 可靠性来自结构组合，而不是单个词：三推、重叠、通道收敛、高潮衰竭、靠近支撑阻力、第三次推动后反向突破。
- 原文强调“楔形顶/底通常至少给出 10 根 K 线、两波”的最小合理目标，本实现据此把目标位设计成 `0.8 倍楔形高度` 与 `1.8R` 的较大者。
- 回测结果表明：广义楔形的信号非常多，但在固定名义仓位和成本约束下，净值仍偏弱，说明课程里的“宽泛楔形”更适合作为研究母集，而非直接交易清单。

# 形态归纳

| 原文线索 | 归一化模式名 | 主/次模式 | 为什么归到这里 |
| --- | --- | --- | --- |
| “任何具有三波推动的形态” | 广义楔形 | 主模式 | 这是本文对楔形的核心定义，比传统收敛楔形更宽。 |
| “楔形顶”“更高高点主要趋势反转” | 楔形顶反转 | 主模式 | 多头趋势中的三推衰竭，目标是向下两波或测试通道底部。 |
| “楔形底”“更低低点趋势反转” | 楔形底反转 | 主模式 | 空头趋势中的三推衰竭，目标是向上两波或测试通道顶部。 |
| “楔形熊旗”“楔形牛旗” | 楔形旗形 | 次模式 | 原文把楔形既看成反转形态，也看成更大级别趋势里的回调结构。 |
| “重叠”“收敛线”“高潮”“FOMO / 拥挤交易” | 交易区间 + 高潮衰竭 | 次模式 | 三推只是外形，真正提高反转概率的是拥挤动量和结构收缩。 |

# 量化定义

## 模式 1：楔形顶

| 字段 | 内容 |
| --- | --- |
| 所需字段 | `timestamp/open/high/low/close/volume` |
| 衍生特征 | `atr`、`ema20`、`ema50`、`uptrend_score`、`buy_climax_count`、`overlap_ratio`、`compression_ratio`、`wedge_high`、`wedge_low`、`up_first_push_idx`、`up_second_push_idx`、`up_third_push_idx` |
| 判定逻辑 | 以 `5` 根回看高点识别向上突破；三次向上推动之间至少间隔 `3` 根 K 线，并要求中间出现至少 `0.35 * ATR` 的回撤；最近 `40` 根内保留最近三次推动；同时要求 `uptrend_score >= 12`、`overlap_ratio >= 0.45`、`compression_ratio <= 0.90`，且满足 `near_resistance` 或 `buy_climax_count >= 4`。 |
| 确认条件 | 第三次推动后 `6` 根内，出现收盘跌破前一根低点、位于 `ema20` 下方、`close_pos <= 0.45` 的空头触发 K。 |
| 失效条件 | 做空后重新上破第三推高点并触发止损。 |
| 待验证假设 | `breakout_lookback=5`、`push_pullback_atr=0.35`、`compression_ratio_max=0.90` 属于经验阈值；不同品种和周期下需要单独校准。 |

## 模式 2：楔形底

| 字段 | 内容 |
| --- | --- |
| 所需字段 | `timestamp/open/high/low/close/volume` |
| 衍生特征 | `atr`、`ema20`、`ema50`、`downtrend_score`、`sell_climax_count`、`overlap_ratio`、`compression_ratio`、`wedge_high`、`wedge_low`、`down_first_push_idx`、`down_second_push_idx`、`down_third_push_idx` |
| 判定逻辑 | 以 `5` 根回看低点识别向下突破；三次向下推动之间至少间隔 `3` 根 K 线，并要求中间出现至少 `0.35 * ATR` 的反抽；最近 `40` 根内保留最近三次推动；同时要求 `downtrend_score >= 12`、`overlap_ratio >= 0.45`、`compression_ratio <= 0.90`，且满足 `near_support` 或 `sell_climax_count >= 4`。 |
| 确认条件 | 第三次推动后 `6` 根内，出现收盘突破前一根高点、位于 `ema20` 上方、`close_pos >= 0.55` 的多头触发 K。 |
| 失效条件 | 做多后重新下破第三推低点并触发止损。 |
| 待验证假设 | “广义楔形”把很多宽通道和嵌套三推也纳入统计，信号密度会显著提高，也更容易把非反转结构混进来。 |

# 交易语义

## 楔形顶

- 入场触发：第三推之后出现反向空头触发 K，做空。
- 止损位置：第三推高点外加 `0.15 * ATR`，且最小止损不低于 `0.7 * ATR`。
- 目标位/测量目标：`max(0.8 * 楔形高度, 1.8R)`，对应“至少两波下行 / 测试通道底部”。
- 失败条件：触发后重新站回第三推高点。
- 适用品种或周期假设：当前只验证 `ETHUSDT 5m`，更低波动市场大概率需要更严格过滤。
- 建议统计指标：三推后的反向突破成功率、空头触发后 10/20/40 根的 MFE/MAE、净值回撤、交易成本敏感性。

## 楔形底

- 入场触发：第三推之后出现反向多头触发 K，做多。
- 止损位置：第三推低点下方减去 `0.15 * ATR`，且最小止损不低于 `0.7 * ATR`。
- 目标位/测量目标：`max(0.8 * 楔形高度, 1.8R)`，对应“至少两波上行 / 测试通道顶部”。
- 失败条件：触发后重新跌破第三推低点。
- 适用品种或周期假设：更适合本身有动量拥挤和反转加速的品种；横盘品种容易把很多噪声波动误识别成楔形。
- 建议统计指标：第三推后的反抽成功率、做多后两波推进比例、持仓分布、净值稳定性。

# 交易策略

| 项目 | 规则 |
| --- | --- |
| 方向 | 楔形顶做空，楔形底做多 |
| 入场规则 | 信号 K 收盘即入场，不额外等待二次确认 |
| 出场规则 | 命中止损、命中目标位，或持仓满 `40` 根 K 线平仓 |

| 过滤条件 | 三推、重叠、局部收缩、趋势得分、高潮计数、支撑阻力 proximity |

# 验证与风险

- 当前实现忠实采用了课程里的“广义楔形”定义，因此把很多三推通道、嵌套楔形、宽通道中的三次推动也算进来，信号数会非常大。
- `label_wedge_outcomes()` 使用未来 `10` 根 K 线，只能用于研究，不可用于实时交易。
- 当前样本只覆盖 `ETHUSDT 5m`，没有做年份切分和样本外验证；下一步应按年份或滚动窗口验证。
- 课程里“75% 概率得到反向突破”是主观经验，本实现并未在该样本上复现实盘可用的净值表现，应视为待验证假设。
- 若要继续研究，优先考虑收紧结构过滤：例如只保留更明显的收敛楔形、强支撑阻力附近的第三推、或配合更严格的反向突破质量过滤。
