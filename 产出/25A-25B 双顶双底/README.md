# 文档摘要

- `25A` 与 `25B` 把双顶双底定义得很宽：它既可以是反转形态，也可以是趋势中的旗形或小型交易区间。
- 原文强调“测试前高/前低”而不是几何上完全对称，因此本实现允许略创新高的双顶、略更低低的双底，以及略高低/略低高的次级变体。
- 课程里的核心交易语义有三层：前高前低测试、失败后的反向交易、以及基于顶底到颈线高度的等距运动目标。
- 量化时不能只给布尔识别；还要抽出肉眼真正依赖的关键感知点，例如两个顶/底、颈线点、触发点，并在案例图里简单连线。
- 全样本回测显示：宽定义双顶双底会产生大量机会，但直接交易噪声很高，固定名义仓位与成本后净值明显为负，更适合作为研究母集。

# 交易语义

## 双顶

- 入场触发：第二个顶之后出现空头触发 K，做空。
- 止损位置：参考顶点外加 `0.12 * ATR`，且最小止损不小于 `0.65 * ATR`。
- 目标位/测量目标：`max(0.75 * 形态高度, 1.8R)`。
- 失败条件：重新站回参考顶点上方，并形成失败突破。
- 适用品种或周期假设：当前只验证 `ETHUSDT 5m`。
- 建议统计指标：双顶 setup 数、信号数、等距运动命中率、失败突破率、交易成本敏感性。

## 双底

- 入场触发：第二个底之后出现多头触发 K，做多。
- 止损位置：参考底点下方减去 `0.12 * ATR`，且最小止损不小于 `0.65 * ATR`。
- 目标位/测量目标：`max(0.75 * 形态高度, 1.8R)`。
- 失败条件：重新跌破参考底点下方，并形成失败跌破。
- 适用品种或周期假设：当前只验证 `ETHUSDT 5m`。
- 建议统计指标：双底 setup 数、信号数、等距运动命中率、失败跌破率、持仓分布。

# 交易策略

| 项目 | 规则 |
| --- | --- |
| 方向 | 双顶做空，双底做多 |
| 入场规则 | 触发 K 收盘即入场，不等待第二次确认 |
| 出场规则 | 命中止损、命中目标位，或持仓满 `36` 根 K 线离场 |
| 止损规则 | 参考顶/底外侧 `0.12 * ATR`，且保证最小风险 `0.65 * ATR` |
| 止盈/目标位规则 | `max(0.75 * 形态高度, 1.8R)` |
| 过滤条件 | 形态跨度 `4-60` 根 K 线，形态高度 `0.9-12 ATR`，触发点需满足收盘位置与均线方向约束 |
| 风险管理假设 | 固定名义仓位 `10000 USDT`，每边 `2 bps` 含手续费与滑点，不做复利 |
| 建议回测指标 | 交易笔数、胜率、平均盈亏、总盈亏、平均 `R`、总 `R`、最大回撤、Profit Factor、净值 |

# 验证与风险

- `label_double_top_bottom_outcomes()` 使用未来 `24` 根 K 线，只能用于研究和标签统计，不能用于实时交易。
- 课程里“略创新高仍算双顶”“略更低低仍算双底”的口径很宽，会显著增加信号密度，也会引入更多弱结构。
- 当前只验证 `ETHUSDT 5m`，没有做年份切分或滚动样本外检验，后续应按年份或 regime 切片重新验证。
- 当前策略没有区分“反转型双顶双底”和“旗形型双顶双底”的不同出场逻辑，这会混淆统计。
- 净值已跌破零，说明固定名义仓位回测在当前假设下不具备可上线性；后续应优先收紧定义，而不是继续放大交易数量。
