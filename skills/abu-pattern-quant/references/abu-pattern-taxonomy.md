# 阿布价格行为形态族

这个文件用于把阿布课程语料中的自然语言归并为稳定的量化模式族。先归类，再量化；不要直接对原文逐句翻译。

## 统一原则

- 先识别结构，再识别交易语义。
- 先找 swing、高低点测试、通道边界、交易区间边界、突破与失败突破，再讨论入场止损目标。
- 同一个描述可以同时属于多个层级，例如“双顶熊旗”同时属于 `双顶`、`旗形`、`空头延续`。
- 文本中的“微型”“小型”“宽阔”“紧密”“强劲”都必须转成阈值或待验证假设。

## 标准模式族

| 模式族 | 常见别名/线索 | 默认量化抓手 | 常见交易语义 |
| --- | --- | --- | --- |
| 趋势 | 多头趋势、空头趋势、always in long/short、趋势恢复 | 连续更高高/更高低或更低低/更低高；均线斜率；连续收盘方向；回调深度 | 顺势入场、回调结束恢复、首次逆势反转多为失败 |
| 通道 | 牛通道、熊通道、窄通道、微通道、宽阔通道 | 通道边界回归线；触边次数；斜率；通道宽度/ATR；单边连续 K 线数 | 通道内回调顺势；紧通道里首个反转多为旗形 |
| 交易区间 | 横盘、盘整、震荡、共识区、磁铁 | 局部高低点重叠；区间高度/ATR；往返次数；突破后回归速度 | 区间上沿/下沿反转；突破后失败与回归 |
| 突破 | 突破、强突破、缺口突破、测量移动 | 突破幅度/ATR；突破 K 实体占比；跟随 K 数量；收盘位置；缺口是否关闭 | 突破后跟进、突破回踩、失败突破反向交易 |
| 双顶双底 | 测试前高、测试前低、略创新高/低、颈线 | 两次测试间距；高低点价差阈值；颈线位置；第二次测试后的确认 K | 反转或旗形；入场在颈线突破/失败确认；测量目标取颈线到顶底距离 |
| 三角形 | 收缩三角形、上升三角形、下降三角形、紧凑交易区间 | 至少三次同向反转；上下边界收敛；区间持续 bar 数；假突破次数 | 突破方向选择；首个突破失败概率；末端常为最后旗形 |
| 楔形 | 三推、楔形顶/底、楔形旗、楔形通道 | 三次推动；推动间距；每次推动动能减弱；倾斜边界 | 倾斜方向的相反突破概率更高；常与主要趋势反转相连 |
| 趋势反转 | MTR、主要趋势反转、更高低/更低高反转 | 趋势后横盘；失败趋势恢复；二次入场；关键 swing 失守 | 入场重视确认与失败突破；止损放在关键 swing 外 |
| 支撑阻力 | 前高前低、磁铁、边界、突破点回测 | 水平价位反复触碰次数；价差容忍度；回踩/回抽成功率 | 回测做多/做空；边界失守转为反向目标 |
| 高潮/衰竭 | 买入高潮、卖出高潮、加速、衰竭缺口 | 连续大实体 K；通道斜率突增；距离均值过大；成交量异常 | 高潮后先看交易区间或回撤，不直接假设大反转 |

## 优先提取的结构要素

- `swing_high` / `swing_low`：局部高低点，决定测试、双顶双底、边界和颈线。
- `leg_count`：推动次数，决定楔形、三角形、高潮后的二次尝试。
- `overlap_ratio`：K 线重叠程度，辅助识别交易区间和紧凑形态。
- `channel_width_atr`：通道宽度相对 ATR 的比例，区分紧通道与宽通道。
- `breakout_strength`：突破 K 实体、收盘位置、后续跟随 K 组成的强度分数。
- `retest_distance`：第二次测试与第一次测试的价差，用于双顶双底和支撑阻力测试。

## 主观词到量化的翻译原则

| 原始词 | 不可直接输出 | 可用替代量化方式 |
| --- | --- | --- |
| 强/弱 | “强势上涨” | 连续阳线数、实体占比、收盘靠近高点、突破幅度/ATR |
| 明显 | “明显突破” | 高于边界至少 `x * ATR` 或 `x%`，并有 `n` 根跟随 K |
| 测试 | “测试前高” | 第二次高点距参考高点在容忍区间内 |
| 接近 | “接近前低” | 价差不超过容忍阈值，例如 ATR、tick 或百分比 |
| 失败 | “失败突破” | 突破后 `n` 根内回到边界内，且向反方向收盘确认 |
| 紧密/宽阔 | “紧通道” | 通道宽度、回调深度、对边界偏离度相对 ATR 的约束 |

## 校准建议

- 优先在单一品种、单一周期上校准阈值，不要跨市场直接复用。
- 先用样本数据做可视化抽样，再确定阈值，不要先拍脑袋定数。
- 如果原文只给方向性语义，先输出待验证假设，再建议用分位数、ATR 倍数或 swing 统计来定阈值。
